webStorm 配置

  1. 点击 File → Settings → Languages & Frameworks → JavaScript → Code Quality Tools → ESLint

  2. 点击 File → Settings → Keymap → 搜索 fix → 选中JavaScript and TypeScript → Fix ESLint Problems → 右键 KeyBoard Shortcut 设置按键

  3. 点击 File → Settings → Editor → Code Style → JavaScript （设置缩进）
